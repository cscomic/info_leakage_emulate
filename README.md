# AI 测试项目

## 项目概述

这是一个用于测试代码安全和合规性检查的虚拟 Python AI 项目。主要目的是在提交到 GitHub 后，通过安全巡检平台检测代码中可能存在的问题模式，验证代码扫描工具的有效性。

## 项目结构

```
src/
├── com/
│   ├── __init__.py
│   └── isstech/
│       ├── __init__.py
│       └── fin/
│           ├── __init__.py
│           ├── test_code.py           # 包含各种测试模式的代码
│           ├── security_patterns.py   # 安全相关代码模式
│           └── ai_models.py          # AI模型相关代码
tests/
├── __init__.py

.gitignore
README.md

```

## 使用说明

### 环境设置
1. 克隆项目到本地
2. 创建虚拟环境：`python -m venv venv`
3. 激活虚拟环境：
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. 安装依赖：`pip install -r requirements.txt`

### 运行测试
```bash
# 运行所有测试
pytest tests/

# 运行特定测试文件
pytest tests/test_functions.py
```
